간단하게 코드 조각 돌리기 좋은 서비스


Wandbox : 여러 언어의 컴파일, 컴파일 옵션 및 실행(표준 입출력)을 지원하며 파일 분할 컴파일도 가능. 자세한 내용은 여기로.


Ideone : Wandbox와 비슷하지만 링크로 코드 공유가 가능. 자세한 내용은 여기로.

TIO : 심플한 디자인이 매력이지만 지원하는 언어는 매우 많습니다.

Compiler Explorer : 언어별, 컴파일러별로 컴파일된 어셈블리 코드를 쉽게 볼 수 있음.


Colaboratory : 파이썬 주피터 노트북을 아시는가? 그거의 온라인 버전이라고 보면 된다! 머신러닝 공부하기 좋다!

JS Fiddle : HTML, JS, CSS 등을 이용해 웹 개발 가능. 여러 라이브러리와 boilerplate들을 지원합니다.

WebAssembly Studio : C, Rust를 컴파일해서 웹 어셈블리 코드를 만들고 이를 웹 환경에서 테스트할 수 있다.


본격적인 개발 도구나 서비스


VS Code : 사지방 컴은 32비트라서 전용 설치파일을 따로 써야 함! 확장 프로그램을 깔 수는 있지만 매번 귀찮으니 편집기로만 쓰자.

Codeanywhere : 코드 뿐만 아니라 리눅스 터미널(!)도 사용할 수 있고 계정에 코드가 저장된다! 원격지에서 접속 가능한 주소도 제공한다.

구름IDE : 이름 그대로 IDE고 Codeanywhere처럼 다양한 컨테이너와 터미널을 지원한다. 협업에 쓰는 것도 가능!

Paperspace : 클라우드 호스팅. 사실상 이것만 있어도 궁극의 개발 환경을 구축할 수 있습니다. 본인 서버가 있다면 더 좋지만... 자세한 내용은 여기로.


코드 관리하기 좋은 서비스

Git + GitLab : 단순히 업로드만 할꺼라면 클라우드로도 상관없지만 요즘 버전 컨트롤은 필수! GitHub이 안돼서 찾은거지만 더 좋습니다.

기타 유용한 도구나 서비스


Pixlr : 온라인 포토샵! 개발할때 이미지 편집이 필요한 경우엔 이 사이트로!

Trello : 이슈 트래커! 개인 메모로 사용하기에도 좋고 작은 용량의 파일은 업로드/다운로드도 가능!

Discord : 메신저다! 접속이 가능하고 잘 되는 메신저는 이거! 여러 유용한 챗봇을 이용하여 개발에 사용할 수 있다. 아님 직접 만들거나!

Online Development Tools : 말 그대로 이 글의 주제와 같이 온라인 개발을 위한 도구가 잔뜩! 있습니다.

draw.io : 큰 프로젝트를 개발하려면 코드 구조를 설계할 일도 있죠. 웹 드라이브와 연동되는 다이어그램 사이트!

mLab : MongoDB 기반으로 0.5GB의 DB를 무료로 제공하며 웹 페이지에서 조회/수정 가능. 개발자라면 유용하게 쓸 수 있을겁니다.

RegExr : 정규식 테스트 사이트. 자세한 내용은 여기로.
